# Progress Update Template

Use this format for recurring build updates:

## Status Update
- current focus:
- completed since last update:
- current blocker (if any):
- next step:

## Summary
Write 3-6 concise bullets describing what changed in plain language.

## Guidance
- keep it short enough for Telegram
- mention only meaningful progress
- if there is no meaningful progress, say what is waiting or what is being investigated
- after sending the update, continue work autonomously
